# ipre-aplicacion-neural-sa
Repositorio para ICS2985. 
Experiencia computacional para verificar funcionamiento y robustez de Neural Simulated Annealing (inspirado en https://doi.org/10.48550/arXiv.2203.02201). 

Elaborado por Isidora Carmona, Francisca Correa, Javiera Santibañez y Sebastian Sepúlveda.
Bajo supervisión de Jorge Vera.
